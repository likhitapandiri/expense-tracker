 NestJS:

A structured backend framework on top of Node.js (like Spring Boot in Java)

Key ideas:
Uses modules, controllers, services
Built on Express/Fastify
Encourages clean architecture

Module is a logical container that groups related providers, controllers, and imports.
What exists (providers)
What is exposed (exports)
What it depends on (imports)
----------------------------------------------------------------
Prisma :
is an ORM (Object Relational Mapper)

Meaning:
Instead of writing raw SQL:

SELECT * FROM transactions;

You write:
prisma.transaction.findMany()

What Prisma gives you:
Type safety (VERY important)
Auto-generated queries
Schema-driven DB design - Your database structure is defined in one central schema file, and everything (tables, queries, types) is generated from it.

prisma (CLI)
Runs in terminal
Used for setup

Prisma Client
Used in your code
Talks to DB

---------------------------------------------------------------------------------
SQL (Structured Query Language) is a language

It’s like:
A way to “talk to databases”

Example:
SELECT * FROM Transaction;

👉 Meaning:
“Give me all transactions”
